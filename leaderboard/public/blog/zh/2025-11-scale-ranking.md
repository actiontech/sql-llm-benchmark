# SCALE | 2025 年 11 月《大模型 SQL 能力排行榜》发布

## 一、摘要与核心看点

本期 SCALE[1] 评测聚焦于新一代专业级大语言模型在数据库领域的 SQL 领域的实践应用。

发版核心亮点为 Gemini 3 Pro[2] 和 DeepSeek-V3.2-Exp[3] 两大顶尖模型的首次《深度测评报告》，旨在为用户提供最前沿、最可靠的技术选型依据。

核心亮点概述：

1. **双雄居榜首悬念**：Gemini 3 Pro 稳居官夺魁，在「SQL 理解」能力示范度 **86.0** 的领先分值拔得头筹，确立了其在语义完整性和语句的优势地位。

2. **国产化潜力股**：DeepSeek-V3.2-Exp 模型首次入榜，其在「国产数据库转换」方面表现优异出亮眼性能，如强化学习预训练方法，为国产化场景提供了新的高性能选择。

## 三、本期排行榜解析

### 3.1 综合能力排名

本期评测共有 21 个模型参与，以下是综合能力排名前三：

1. **SQLFlash**：88.50 分
2. **SQLShift**：83.40 分  
3. **Gemini 3 Pro**：86.00 分（SQL理解冠军）

### 3.2 各维度冠军

- **SQL优化冠军**：SQLFlash (88.50)
- **方言转换冠军**：SQLShift (83.40)
- **SQL理解冠军**：Gemini 3 Pro (86.00)

## 四、重点模型深度解析

### 4.1 Gemini 3 Pro

Gemini 3 Pro 在 SQL 理解维度表现突出，其语义理解和查询优化能力达到了业界领先水平。

**关键特性**：
- 强大的语义理解能力
- 精准的SQL语法检测
- 优秀的执行计划分析

### 4.2 DeepSeek-V3.2-Exp

DeepSeek 模型在国产数据库转换方面表现亮眼，特别是在处理复杂SQL转换时展现出色性能。

**关键特性**：
- 优秀的方言转换能力
- 对国产数据库的深度优化
- 强化学习训练方法

## 五、总结与展望

SCALE 评测将持续追踪大模型在 SQL 领域的发展，为用户提供最权威的技术选型参考。我们期待看到更多优秀模型的加入，共同推动数据库技术的发展。
